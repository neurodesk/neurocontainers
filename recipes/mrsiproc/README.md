## mrsiproc/0.2.0 ##

This container provides the MR Spectroscopy Imaging Processing Pipeline with LCModel, FSL, MINC, HD-BET, Julia, FreeSurfer, dcm2niix, and MATLAB Runtime components.

Example:
  bash Part1_ProcessMRSI.sh
  bash Part2_EvaluateMRSI.sh

To run container outside of this environment: ml mrsiproc/0.2.0
