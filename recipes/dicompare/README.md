# dicompare

dicompare is an MRI protocol validation tool that helps researchers validate their DICOM files against community-defined acquisition protocols.

## Usage

Launch the webapp in Neurodesk JupyterLab, then drag and drop your DICOM files to validate them against protocol definitions.
